# Vigenere Cipher (C)
Программа шифрует и дешифрует текст с помощью шифра Виженера.

Как работает:
- Приводит текст и ключ к нижнему регистру.
- Удаляет пробелы.
- Строит таблицу Виженера.
- Шифрует и расшифровывает текст.

Ограничения:
- Текст ≤ 100 символов.
- Ключ ≤ 10 символов.
- Только английские буквы.

Компиляция:
gcc vigenere.c -o vigenere
./vigenere
