# README

## Описание
Программа — простая модель шифратора с 3 роторами (аналог Enigma).  
Шифрует текст из латинских букв по заданному ключу.

## Компиляция
gcc enigma.c.c -o enigma

## Запуск
./enigma

## Ввод
1. Текст (только буквы a–z).  
2. Ключ из 3 букв (например: abc).

## Работа
Каждый ротор меняет буквы по своей таблице.  
Роторы вращаются с разной скоростью (частоты 1, 26, 676).
