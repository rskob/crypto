# Книжный шифр (C)
Программа шифрует и дешифрует текст с помощью гомофонного шифра подстановки.

Как работает:
- Читает ключ из файла key.txt (список слов).
- Для каждой буквы алфавита создаёт список индексов из ключевого файла.
- Шифрует текст: каждую букву заменяет случайным индексом из её списка.
- Дешифрует текст: по индексам восстанавливает исходные буквы.

Особенности:
- Каждая буква может иметь несколько вариантов замены.
- Удаляет все не-буквенные символы при обработке.
- Работает только с английскими буквами.

Требования:
- Файл key.txt с набором слов/букв (каждое слово начинается с новой буквы).
- Текст ≤ 1000 символов.

Компиляция:
gcc book.c -o book

./book
